# Image Slider (JavaScript)

A simple image slider built with HTML, CSS, and JavaScript that allows users to navigate through images using previous and next controls.

## Overview
This project was created to practice JavaScript fundamentals such as DOM manipulation, event handling, and basic animation using CSS.

## Features
- Displays a collection of images in a slider format
- Previous and next navigation buttons
- Automatic slide transition every few seconds
- Fade animation between slides
- Responsive 16:9 layout

## Technologies
- JavaScript
- HTML
- CSS

## Usage
1. Open the `index.html` file in your browser.
2. Use the **previous** and **next** buttons to navigate between images.
3. Slides will also change automatically after a set interval.
